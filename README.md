# MGS 光储充一体化场站配置与运行决策平台

## 软件定位

本软件面向光储充一体化场站的配置与运行决策。系统首先根据车辆需求、光伏储能参数和并网边界生成基准配置；随后在压力测试情形下识别功率越限、储能 SOC 风险、排队损失和未满足电量等故障参数；在此基础上通过分时价格调度引导弹性车辆错峰充电，并进行全年连续验证；最后针对价格调度后仍然存在的残余风险生成工程加固方案，输出推荐配置。

## 当前主链

```
M1 基准配置生成
→ M2 压力测试诊断
→ M3 价格调度与全年验证
→ M4 残余风险加固推荐
```

## S0~S4 方案族

- S0：基准对照
- S1：功率边界加固
- S2：储能韧性加固
- S3：服务能力加固
- S4：综合平衡方案

## 关于 PVUR / GFF

当前版本已经接入可运行的工程近似指标：
- PVUR proxy：基于全年光伏发电与估算弃光量构造
- GFF proxy：基于峰/平/谷购电结构构造
