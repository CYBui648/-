# MGS V2-4：M4 最终工程方案定型器

## 这一版完成了什么
V2-4 将 M4 正式落地为“最终工程方案定型器”：

- 承接用户在 M3 选择的技术路线
- 读取该路线下的残余风险
- 自动生成 S0~S4 五类典型工程方案
- 对每个方案进行：
  - 压力月复验
  - 全年复验
- 计算综合评价指标：
  - 风险修复
  - 追加投资
  - PVUR proxy
  - GFF proxy
  - 年度 LCOE
- 输出：
  - 综合推荐
  - 低投资备选
  - 高保障备选

## S0~S4
- S0：基准对照
- S1：功率瓶颈加固
- S2：储能韧性加固
- S3：服务能力加固
- S4：综合平衡方案

## 关于 PVUR / GFF
当前版本已经接入可运行的工程近似指标：
- PVUR proxy：基于全年光伏发电与估算弃光量构造
- GFF proxy：基于峰/平/谷购电结构构造

这样做的目的，是先把 M4 的“方案生成 → 复验 → 综合推荐”主链跑通。
后续若要严格对齐文献公式，可以再精修这两个指标口径。

## 当前主链
M1 真实规划
→ M2 真实压力测试
→ M3 双路线并列评估
→ 用户选择路线
→ M4 S0~S4 工程方案定型
